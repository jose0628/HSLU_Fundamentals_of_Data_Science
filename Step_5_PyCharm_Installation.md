# PyCharm Professional Installation Guide

A step-by-step guide for downloading, installing, and activating the **PyCharm Professional Edition** (“Pro” version). 

---

## 1. System Requirements

Make sure your system meets the minimum requirements to run PyCharm Professional:

- **Operating System**: Windows, macOS, or Linux.
- **RAM**: 2 GB (minimum); 8 GB recommended for improved performance.
- **Disk Space**: 2.5 GB (minimum) plus at least 1 GB for caches/indexes.
- **Python Interpreter**: Python 2.7 or Python 3.x. Anaconda.

For the most accurate details, refer to the [official JetBrains website](https://www.jetbrains.com/pycharm/).

---

## 2. Download PyCharm Professional

1. **Go to the Download Page**  
   Visit [https://www.jetbrains.com/pycharm/download/](https://www.jetbrains.com/pycharm/download/).

2. **Select Professional Edition**  
   You’ll see two editions: **Professional** and **Community**. Choose **Professional** to access all “Pro” features.

3. **Choose Your OS**  
   Select your operating system: Windows, macOS, or Linux.

4. **Download**  
   Click the **Download** button under **Professional** to start downloading the installer.

---

## 3. Installation Steps

### A. Windows

1. **Locate Installer**  
   The `.exe` file is usually saved to your **Downloads** folder.

2. **Run Installer**  
   Double-click the `.exe` to launch the setup wizard.

3. **Setup Wizard**  
   - Specify the **installation path** (default: `C:\Program Files\JetBrains\PyCharm <version>`).  
   - Select additional options (desktop shortcut, “Open Folder as Project” context menu, file associations, etc.).

4. **Install**  
   Click **Install** and wait for completion.

5. **Finish**  
   Optionally check **Run PyCharm** at the end, then click **Finish**.

### B. macOS

1. **Open the DMG**  
   The downloaded `.dmg` file is typically in the **Downloads** folder. Double-click to open it.

2. **Drag & Drop**  
   Drag the **PyCharm** icon into the **Applications** folder as instructed.

3. **Launch PyCharm**  
   - Open the **Applications** folder and double-click **PyCharm**.  
   - If prompted about an unverified developer, click **Open**.

4. **Initial Settings**  
   You can import settings from a previous version if available.
