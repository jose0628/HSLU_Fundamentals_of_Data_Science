# Install Ollama on Windows and macOS

## Windows (10/11)

### 1. Download & Install
- Visit the [Ollama website](https://ollama.com) and download the Windows `.exe` installer.
- Run the installer and follow the setup wizard.
- Ollama will start automatically after installation.

### 2. GUI Application
- Includes a desktop GUI app for chatting with models without the terminal.
- Supports drag-and-drop of files, adjusting settings, and switching models.

### 3. Running a Model
- Open the Ollama application.
- Use the GUI to select and run models such as **llama3.2**.

### 4. Uninstall
- Go to **Settings → Apps → Installed Apps** and remove Ollama.
- Delete `%HOMEPATH%\.ollama` manually to remove downloaded models.

---

## macOS (Intel or Apple Silicon)

### 1. Download & Install
- Visit the [Ollama website](https://ollama.com) and download the `.dmg` installer.
- Open the `.dmg` and drag **Ollama** to **Applications**.

### 2. First Launch & Permissions
- On first launch, macOS may warn about the source. Click **Open**.
- Ollama runs in the background and adds an icon to the menu bar.

### 3. Running a Model
- Open the Ollama application.
- Use the GUI to select and run models such as **llama3.2**.

### 4. Updating
- Download the latest `.dmg` from the [Ollama website](https://ollama.com) and replace the existing app.
