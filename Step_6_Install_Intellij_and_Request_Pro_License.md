# Step 6: Install IntelliJ IDEA and Request Pro License

Applies to: Windows and macOS

## Goal
Get a free JetBrains student license and activate IntelliJ IDEA Ultimate.

## A. Check Eligibility
1. Confirm you are an active HSLU student.
2. Prepare one of the following:
   - HSLU student email address.
   - Proof of enrollment document.

## B. Create a JetBrains Account
1. Open https://www.jetbrains.com/
2. Select **Sign In** then **Create Account**.
3. Complete email verification.

## C. Request the Student License
1. Open https://www.jetbrains.com/community/education/#students
2. Select **Apply Now**.
3. Choose one verification method:
   - School email (fastest when domain is recognized).
   - ISIC card.
   - Enrollment proof upload.
4. Submit and wait for approval email.

## D. Install IntelliJ IDEA Ultimate
1. Open https://www.jetbrains.com/idea/download/
2. Download **Ultimate** for your operating system:
   - Windows: `.exe` installer.
   - macOS: `.dmg` installer (Apple Silicon or Intel as appropriate).
3. Install and start IntelliJ IDEA.
4. Sign in with your JetBrains account to activate the student license.

## E. Verification
1. In IntelliJ IDEA, open **Help -> Register** (or license status view).
2. Confirm license type is student and expiration is visible.

## Renewal
- Student licenses are typically valid for one year.
- Renew before expiry from the same student page.

## Troubleshooting
- School email rejected: use enrollment proof.
- No approval email: check spam/junk and JetBrains account notifications.
- App still unlicensed: sign out/in inside IntelliJ and restart once.
